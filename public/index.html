Finish setup

Almost there — add your GitHub OAuth App credentials to Netlify and you're live.

1. Go to github.com/settings/developers
2. Click New OAuth App
3. Set Homepage URL to your Netlify URL
4. Set Callback URL to https://YOUR-SITE.netlify.app/oauth/callback
5. In Netlify → Site settings → Environment variables, add:
  GITHUB_CLIENT_ID and GITHUB_CLIENT_SECRET
6. Redeploy — done!

The DEPLOY.md in your downloaded zip has full instructions.
